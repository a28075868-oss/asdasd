ЗАГРУЗКА И ЯДРО:
  ✓ multiboot.asm      - Multiboot заголовок, точка входа (Assembly)
  ✓ kernel.c/h         - Главное ядро системы (C)
  ✓ linker.ld          - Скрипт компоновщика

УПРАВЛЕНИЕ ПАМЯТЬЮ:
  ✓ gdt.c + gdt.asm    - Global Descriptor Table (C + Assembly)
  ✓ memory.c/h         - Динамическое выделение памяти (kmalloc/kfree)

ПРЕРЫВАНИЯ:
  ✓ idt.c/h            - Interrupt Descriptor Table (C)
  ✓ interrupt.asm      - Обработчики прерываний ISR/IRQ (Assembly)

ДРАЙВЕРЫ:
  ✓ keyboard.c/h       - Драйвер клавиатуры PS/2 с буфером
  ✓ timer.c/h          - Программируемый таймер (PIT)
  ✓ screen.c/h         - VGA текстовый режим 80x25, цвета

ПРОЦЕССЫ И СИСТЕМНЫЕ ВЫЗОВЫ:
  ✓ process.c/h        - Управление процессами, планировщик
  ✓ syscall.c/h        - Системные вызовы (write, read, exit, getpid)

ПОЛЬЗОВАТЕЛЬСКИЙ ИНТЕРФЕЙС:
  ✓ shell.c/h          - Интерактивная командная оболочка

УТИЛИТЫ:
  ✓ string.c           - strlen, strcmp, strcpy, memset, memcpy
  ✓ types.h            - Базовые типы данных (uint8, uint16, uint32, etc.)

Entrada OS создана как учебная операционная система для:

  • Изучения архитектуры x86
  • Понимания работы ОС на низком уровне
  • Практики программирования на C и Assembly
  • Разработки драйверов устройств
  • Изучения управления памятью и процессами
  • Понимания загрузки и инициализации ОС

┌─────────────────────────────────────────────────────────────────────────────┐
│ 🚀 КАК НАЧАТЬ                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

1. Откройте START_HERE.txt
2. Выберите способ сборки (WSL2 / Docker / Онлайн)
3. Соберите проект (make или run_docker.bat)
4. Запустите entrada.iso в QEMU/VirtualBox/VMware
5. Наслаждайтесь своей ОС!

Система спроектирована для легкого добавления:

  • Новых драйверов (мышь, звук, сеть)
  • Файловой системы (FAT, ext2)
  • Виртуальной памяти (paging)
  • Многозадачности (preemptive scheduling)
  • Пользовательского режима (ring 3)
  • Системных утилит
  • Графического режима (VGA/VESA)
  • Сетевой поддержки (TCP/IP)
